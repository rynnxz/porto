# Portfolio — Akbar Riansyah

Portfolio website pribadi yang menampilkan proyek, pengalaman, dan keahlian saya sebagai full-stack developer.

## Tech Stack

- **HTML5** & **CSS3**
- **Bootstrap 5.3.8** - Framework CSS
- **AOS (Animate On Scroll)** - Scroll animations
- **Typed.js** - Typing animation effect
- **Simple Icons** - Technology logos

## Features

- Glass morphism design dengan gradient purple-blue
- Scroll reveal animations pada setiap section
- Typing animation pada hero text
- Hover effects pada project cards dan skill badges
- Interactive profile panel dengan image reveal
- Fully responsive design

## Project Structure

```
porto/
```
